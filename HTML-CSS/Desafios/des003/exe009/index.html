Trabalhando com Listas

Listas Ordenadas

Para criar uma lista onde a ordem importa, usamos a tag ol e depois li para cada item, como abaixo

  1. Acordar
  2. Lavar o Rosto
  3. Entrar na aula
  4. Abrir o Clash
  5. Almoçar
  6. Trocar de roupa
  7. Ir para o trabalho

Podemos também mudar o estilo, com type após o comando ol

  1. Acordar
  2. Lavar o Rosto
  3. Entrar na aula
  4. Abrir o Clash
  5. Almoçar
  6. Trocar de roupa
  7. Ir para o trabalho

Listas não ordenadas

Para listas que não precisam de ordem, usamos ul

  • Pão
  • Macarrão
  • Arroz

Existem outros tipos também das não ordenadas

  • Arroz
  • Feijão
  • Carne

Juntando listas

Podemos também misturar listas ordenadas e não ordenadas

  1. Jogos
    • The Last of us 2

  2. Filmes
